Detailed Description

Sluzi na nacitavanie wav suboru. (managed C++).

Drzi si ulozenu aktualnu poziciu v subore a pri zavolani metody GetCurrentSample(chann) vrati vzorku z tejto pozicie zo zvoleneho kanalu. Pre posun na dalsiu poziciu je potrebne volat metodu step().
